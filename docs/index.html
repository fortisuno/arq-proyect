Descripción

Este es un proyecto para la materia de Arg. y Org. de Computadoras impartida por el profesor Contreras Velasco Jose Antonio en la secuencia 2NM31.

Se trata de un sistema que simula ser un espejo otorgando funciones extras como lo es interactuar con un asistente virtual, por ejemplo, Google Assistant. Además de ello podra hacer funciones como IoT con el fin de utilizar un centro de control para casas inteligentes con un diseño más amigable y discreto para el lugar donde sea utilizado.
